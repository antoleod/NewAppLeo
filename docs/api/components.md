# UI Components Documentation

This document provides an overview of the custom UI components available in AppLeo.

## Components

### Theme Components
- **ThemeCustomizer**: Theme selection and customization interface
- **ThemeQuickSettings**: Quick theme switching widget
- **DataImporter**: JSON data import interface
- **BackgroundPhotoSelector**: Background photo upload and management

### Base UI Components
- **Button**: Primary action button with theme support
- **Card**: Container component with elevation and spacing
- **Toggle**: Animated on/off switch
- **ColorSwatch**: Color display component
- **SectionHeader**: Section title with optional actions
- **ButtonGroup**: Grouped button layout
